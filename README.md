# 概要
YMM4 射出エフェクトプラグイン  
画像を一定間隔で射出させる映像エフェクトプラグインです。  
映像エフェクトの「アニメーション」カテゴリに「射出」エフェクトを追加します。

[最新バージョン](https://github.com/tanazun/EjectEffectPlugin/releases/latest)

# 変更履歴
- 2025-09-14　Ver1.4.0 公開
  - 射出する各アイテムの拡大率と回転角をランダムに変更する機能を追加。
  - 射出の時刻を指定する機能を追加。
  - アニメーションにより設定値が既定の範囲を超えたときに、範囲内に収める処理を追加。
  - 変更点をまとめた動画(Xのポスト)：https://x.com/tanazun0/status/1967071059859136785
- 2025-09-07　[設定項目の説明](./USAGE.md)ページ 公開
- 2025-08-24　Ver1.3.0 公開
  - 射出の繰り返し回数を指定する機能を追加。
  - 0周期目を表示する機能を追加。
  - 射出の終了を調整する機能を追加。
  - 周期ごとのばらつきを固定する機能を追加。
  - 変更点をまとめた動画(Xのポスト)：https://x.com/tanazun0/status/1959530896261849151
- 2025-08-15　Ver1.2.0 公開
  - 目標位置などを変化させても、射出済みのものは軌道を保持する機能を追加。(このページに機能の補足があります)
  - 射出位置を変更する機能を追加。
  - 射出・目標位置を極座標形式で指定する機能を追加。
  - 設定項目の分類を整理。
  - 射出数の上限を100個→300個に変更。(多く設定すると処理が重いので注意)
  - 周期の上限を99秒→100秒に変更。
  - 変更点をまとめた動画(Xのポスト)：https://x.com/tanazun0/status/1956014907868242379
- 2025-07-10　Ver1.1.1 公開
  - 射出角度と射出・目標位置のばらつきの大きさが連動していた不具合を修正。
  - 変更点をまとめた動画(Xのポスト)：https://x.com/tanazun0/status/1943301444989522141
- 2025-07-05　Ver1.1.0 公開
  - 射出角度をばらつかせても、重力を調整することで目標位置を通過させる機能を追加。
  - 射出・目標位置をばらつかせる機能を追加。
  - シード値（の生成に使う値）を指定する機能を追加。
  - 変更点をまとめた動画(Xのポスト)：https://x.com/tanazun0/status/1941484781985988649
- 2025-06-12　Ver1.0.0 公開
