# 概要
画像を一定間隔で射出させる映像エフェクトプラグインです。  
映像エフェクトの「アニメーション」カテゴリに「射出」エフェクトを追加します。

<img src="./images/thumbnail.jpg" alt="動画のサムネイル" width="50%" />

立ち絵：akihiyo様

# ダウンロード・インストール
YMM4のプラグインポータルから、ダウンロード・インストールできます。  
直接プラグインをダウンロードする場合は、[最新バージョン](https://github.com/tanazun/EjectEffectPlugin/releases/latest)からダウンロードしてください。

# 利用に当たり
- 商用・非商用問わず利用できます。
- 利用の際のクレジット表記は任意ですが、表記頂けると幸いです。
  - 表記例：「**使用プラグイン：射出エフェクト**」  
  ※「射出エフェクト」という言葉を入れて頂けると分かりやすいです。
- ニコニコ動画に投稿する際の親子登録は任意ですが、本プラグインの紹介動画(sm45081637)を登録頂けると幸いです。

# 変更履歴
- 2026-01-12　Ver2.1.0 公開
  - 射出のばらつき方を指定する機能を追加。
  - 射出と目標座標にアニメーションを設定に合わせて、プレビュー上の制御点を複数表示するようにした。
  - 射出集中度が100%の時に表示順序がばらばらになる不具合を修正。
  - 他の設定項目の状態に合わせて、一部の設定項目を非表示にするようにした。
    - 例：極座標の使用がOFFのときは、極座標の距離と角度の設定項目が非表示になる。
- 2025-12-14　Ver2.0.0 公開
  - 射出したアイテムに個別にエフェクトを付ける機能を追加。  
    この実装は、以下のプラグインを参考にさせて頂きました。
    - [形貼り付け](https://github.com/HAIsanGames813/AddShapeEffect)：個別にエフェクトを適用する処理のすべて。
    - [CustomEffectSelector](https://github.com/dmmo-com-jp/CustomEffectSelector)：個別エフェクトの更新について。
  - 描画処理を見直しによる処理の高速化。特に、シーンアイテム経由だとコマ送りになる現象が解消。
  - 変更点をまとめた動画(Xのポスト)：https://x.com/tanazun0/status/2000038167987249577
- 2025-11-28　Ver1.6.0 公開
  - 射出したアイテムの表示順序(先に射出したアイテムが手前／後に射出したアイテムが手前)を指定する機能を追加。
  - 「射出・目標位置のばらつき」に、射出位置を固定する機能を追加。
  - 「射出・目標位置のばらつき」に、射出と目標位置のばらつき方向を反転させる機能を追加。
  - アイテムの描画設定にある「回転角」「左右反転」による表示の影響を補正する機能を追加。
    - 設定項目に「描画設定の影響を補正」グループを作成。
    - 既存の「拡大率の影響を補正」の設定をこのグループに移動し、「拡大率」に名前変更。
    - 注意：Ver1.5.0以前で、エフェクトを設定したアイテムの「回転角」と「左右反転」を変更していた場合、表示が変化してしまいます。  
      その場合はこの機能をOFFにすると戻ります。
  - YMM4を再起動すると、ばらつき方が変わってしまう不具合を修正。
  - 変更点をまとめた動画(Xのポスト)：https://x.com/tanazun0/status/1994184638789509425
- 2025-10-13　Ver1.5.0 公開
  - プレビュー画面上で射出・目標位置を変更する機能を追加。
  - 射出と射出の間隔を指定する機能を追加。
  - 変更点をまとめた動画(Xのポスト)：https://x.com/tanazun0/status/1977714204149719053
