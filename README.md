# 概要
YMM4 射出エフェクトプラグイン  
画像を一定間隔で射出させる映像エフェクトプラグインです。  
映像エフェクトの「アニメーション」カテゴリに「射出」エフェクトを追加します。

## 射出済みのアイテムの軌道を保持する機能について
Ver1.2.0で追加された本機能は、以下の設定を時間変化させたときに、射出時の軌道を保持する機能です。

- 射出・目標位置
- 射出角度や射出位置のばらつき
- 重力
- 開始・終了位置調整

以下の設定を時間変化させる場合は、本機能は正しく動作しません。

- 周期
- 射出数
- 射出集中度

理由は、フレーム毎に上記設定項目から射出時の軌道を計算しているからです。
